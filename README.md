## 📌 주요 기능
###  📮 네이버 메일 전송
☑️ 상대방에게 메일을 전송합니다. 
###  📬 전송 메일 조회
☑️ 발신 메일 리스트를 조회합니다. 
### 📝 PDF 다운로드
☑️ 작성 메일을 PDF 형식으로 저장합니다. 
###  🫧 해시태그
☑️ 메일 내용에 해시태그를 넣어 수신자가 중요 키워드를 볼 수 있습니다.
